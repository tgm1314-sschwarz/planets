Controller¶

class Controller.Controller[source]¶

Bases: object

Class that is used to control Buttons and Keys. It also places the Textures for the Buttons.

methods:
  • __init__(): initiates all variables used to control everything
  • key_pressed(): method that constantly checks if a key or button gets pressed
  • cam_control(): method that gets called if either the up or down arrow button or the W or S Key gets pressed and that controls the camera depending on the current position it is in and the button or key that got pressed
  • stopped(): Stops the Animation if the P key is pressed
  • light_on_off(): Turns the light either on or off if the L Key or the Lighting Button is pressed
  • button_textures_on(): Places the textures on the buttons, so that they won’t be disabled if the textures get disabled
  • textures_on_off(): Dis or enables the textures if either the T Key or the Textures Button is pressed
  • get_mouse_pos(): Checks the mouse position at every frame
  • increase(): constantly changes the rotation speeds for all planets so that they keep moving
__init__()[source]¶

Sets all the variables to control the Solar System. Such as the rotation speed or if the light or textures are on or off. Also all the textures are loaded here and saved as Objects.

button_textures_on(n)[source]¶

Places the Textures on the buttons, so that they cant be disabled like the planet textures can

Parameters:n – Name of the button that wants to get a texture placed on
cam_control(move)[source]¶

Method to control the camera. This method is called if either the up/down arrow button or W/S is pressed.

Parameters:move – Either “up” or “down”, depending on the current camera position and the button or key pressed.
get_mouse_pos()[source]¶

Gets the current mouse position of every frame, which is used to test if the mouse got clicked over a button

increase()[source]¶

Always increases the different rotation speeds for the different planets, so that they keep rotating

key_pressed(event)[source]¶

Key and Button listener method. Tests if any Key or any button on the screen gets pressed.

Parameters:event – PyGame event Object that has a type and a key value and is used as a key listener
light_on_off()[source]¶

Dis- and Enables the light if either L or the Light Button is pressed

stopped()[source]¶

Method that gets is used to Pause the game.

textures_on_off(n)[source]¶

Places the textures on the Planets and disables them if the T key or the Texture button is pressed.

Parameters:n – Name of the planet that wants to get a texture mapped on.
